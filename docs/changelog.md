## [1.0.0] — 2025-03-11

### Added
- Initial release with 14 pages
- Authentication (Credentials + Google OAuth via NextAuth.js)
- 8 database tables (Neon PostgreSQL + Drizzle ORM)
- 12 API endpoints with full CRUD
- Newsprint editorial design system (light + dark mode)
- Global search (⌘K) across cases, events, users
- Calendar with 5 view modes (Day, Week, Month, Agenda, Timeline)
- Team collaboration: threaded chat, kanban tasks, time tracking
- Cases pipeline view (Intake → Discovery → Motions → Trial → Closed)
- User settings with profile editing
- Route protection middleware
- Responsive layout with mobile sidebar drawer
- Seed data endpoint for all tables
- Docker deployment config
